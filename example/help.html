How do I install the Coupon Printer?

  1. Click here to download the Coupon Printer installer
  2. Click "Run" to start the installation process in Internet Explorer. Click "Save" in Firefox, then run the install file once the download completes
  3. Read and agree to the license agreement to begin installing the Coupon Printer
  4. Once you see "Installation Successful", your coupons will be sent directly to your system's default printer
